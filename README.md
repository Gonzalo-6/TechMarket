# TechMarket
📝 README - TechMarket
📌 Descripción

TechMarket es una aplicación web desarrollada en JavaScript que simula una tienda online de productos tecnológicos. El objetivo del proyecto es mejorar la experiencia de usuario mediante una interfaz dinámica, permitiendo visualizar productos, filtrarlos, añadirlos a un carrito y realizar una compra simulada.

⚙️ Tecnologías utilizadas
HTML5
CSS3
JavaScript (ES6)
LocalStorage
🚀 Funcionalidades
Visualización dinámica de productos
Filtrado por categorías
Carrito de compra con gestión de cantidades
Persistencia del carrito mediante localStorage
Eliminación y vaciado del carrito
Validación de formulario en cliente
Diseño responsive
Mejora visual con imágenes y efectos hover
🧠 Arquitectura

El proyecto sigue una estructura modular separando responsabilidades en distintos archivos JavaScript:

productos.js → modelo de datos
carrito.js → lógica del carrito
filtro.js → renderizado y filtrado
formulario.js → validación
main.js → inicialización
📱 Responsive Design

Se ha implementado diseño adaptable mediante media queries, permitiendo una correcta visualización en dispositivos móviles, tablets y escritorio.

💡 Mejoras implementadas
Uso de programación orientada a objetos
Gestión del estado mediante arrays
Persistencia de datos en navegador
Interfaz más visual mediante imágenes
Feedback al usuario (mensajes y animaciones)

🎯 Conclusión
La aplicación mejora significativamente la experiencia de usuario respecto a una web estática, facilitando la interacción, reduciendo errores y simulando un entorno real de compra online.
